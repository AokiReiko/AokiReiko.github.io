Projects

What and Where: A Context-based Recommendation System for Object Insertion. [arXiv]

2018.9 - 2018.11, Tsinghua-Tencent Joint Laboratory

In this work, we propose a novel topic consisting of two dual tasks: 1) given a scene, recommend objects to insert, 2) given an object category, retrieve suitable background scenes. A bounding box for the inserted object is predicted in both tasks. We build an unsupervised algorithm based on object-level contexts, which explicitly models the joint probability distribution of object categories and bounding boxes with a Gaussian mixture model. Experiments on our newly annotated test set demonstrate that our system outperforms existing baselines on all subtasks under a unified framework.



Marker-based Body and Garment Reconstruction from Multiview RGB videos. [report] [poster] [slides]

2018.7 - 2018.8, Prof. Ronald Fedkiw's group, Stanford University

In this work, given videos of a moving person from multiple calibrated RGB cameras, we present a marker-based method to get a 3D animation for both the person and the garment. In contrast to previous works, our system simultaneously captures the body and the garment as 2 separate surfaces. First, we digitize the garment by triangulating the boundary of scanned pieces. Afterwards, we track the markers across frames to get their 3D locations as a set of linear sparse constraints. We then optimize over an animatable body template, SMPL, to obtain a body model that is accurate both in pose and shape. Finally, we adopt the level set approach to virtually “wear” the garment on the body.



Bezier-based 3D modeling and rendering. [codes]

2017.5 - 2017.6, Fundamentals of Computer Graphics, Tsinghua University

In this work, I built a scene modeled by Bezier surfaces and rendered with Progressive Photon Mapping (PPM). Both the PPM-based render engine and the Bezier-based models were implemented from scratch in pure C++, without access to external libraries (except for OpenCV for image I/O). The toughest yet most memorable step was to add textures for Bezier surfaces (i.e. the leafs, picture, and hairpin), for which I hypothesized and proved a formula. The project eventually earned not only top 2/123 in the class, but also a place in the course gallery.



Bad Encoding Detection by Text Classification and Language Modeling.

2017.7 - 2017.9, EP Intern, Google Inc.

This work aims to detect bad encodings among billions of multilingual texts in production metadata. I designed a hybrid system comprising a supervised classifier (Char-CNN) and an unsupervised language model (Bigram LM). The entire system was finally tested on the production pipeline and achieved high precision. I was approved by the hiring committee for a return intern offer.



Relation Extraction on Free Text for Knowledge-based Question Answering.

2017.10 - 2018.4, Tsinghua Natural Language Processing Group

Question answering has been a long-standing problem in natural language processing. Structured knowledge base and unstructured free text are two primary resources for answering questions. However, limited works have explored the combination of these two information sources in an end-to-end fashion. We adopted text-based relation extraction methods using supporting sentences from Wikipedia to enrich the representation of question and answer under a multi-task learning framework.
